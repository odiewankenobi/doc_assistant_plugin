# Design System — Color Tokens

Documentación de todas las variables de color organizadas en tres niveles:
1. **Primitivos** (`00. Colors`) — paleta base con valores hex directos
2. **Semánticos** (`01. Colors`) — aliases que referencian los primitivos
3. **Componentes** (`02. Colors`) — tokens específicos por componente
